About CORS & production deployment

HubSpot's API doesn't return browser-friendly CORS headers, so a pure browser app cannot call it directly without a small server-side proxy. For a prototype, use Export-only mode and import the generated CSV via HubSpot's bulk import, or stand up a ~20-line Cloudflare Worker / Vercel function that forwards requests to api.hubapi.com.

Define which fields appear on the contact review form and how they map to HubSpot properties. The email field is used to detect duplicates — leave its internal key as email and its HubSpot mapping as email.

Tips on HubSpot property names

Standard contact properties use lowercase internal names: firstname, lastname, email, phone, mobilephone, jobtitle, company, website, lifecyclestage, hs_lead_status. For custom properties, copy the internal name from HubSpot → Settings → Properties → (your property) → "Internal name". Fields with a blank HubSpot mapping are kept locally but not sent.
